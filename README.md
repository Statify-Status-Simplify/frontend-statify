# Statify - Dashboard

## Introduction
The user-facing web dashboard for Statify. It allows users to create new monitors, track URL uptime, and generate integration tokens for IoT devices.

## Local Setup Instructions
1. Prerequisites: Node.js (v18+)
2. Install dependencies: `npm install`
3. Duplicate `.env.example` to `.env` and configure `VITE_API_BASE_URL` to point to the Spring Boot backend.
4. Start the development server: `npm run dev`
